## ğŸš€ Getting Started

### Prerequisites

- Python 3.8 or newer
- pip  â€” usually included with Python
- git (optional, required to clone the repository)
- Recommended: a virtual environment to isolate dependencies (e.g. `python -m venv .venv`)

### Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/HabiburRahmanZihad/NagarNirman.git
   cd NagarNirman
   ```

2. **Set up a virtual environment (Recommended):**

   ```bash
   python -m venv .venv
   # Windows:
   .venv\Scripts\activate
   # Linux/macOS:
   source .venv/bin/activate
   ```

3. **Install dependencies:**

   ```bash
   pip install streamlit pandas fpdf2
   ```

4. **Launch the application:**
   ```bash
   streamlit run app.py
   ```
